# HR Admin Offboarding Execution Checklist

This checklist ensures consistent and compliant execution of the offboarding workflow.

## Exit Initiation
- [ ] Resignation or termination approved
- [ ] Exit type identified
- [ ] Notice period calculated and recorded

## Status & Communication
- [ ] Employee status updated to Exit Initiated
- [ ] Exit communication sent to employee and manager

## Notice Period
- [ ] Notice period dates tracked
- [ ] Handover plan initiated
- [ ] Attendance obligations monitored

## Clearance & Access
- [ ] Asset handover completed
- [ ] System access revoked
- [ ] Clearance approvals recorded

## Final Settlement
- [ ] Final settlement calculated
- [ ] Exit documents generated
- [ ] Employee record closed and archived

## Completion Check
- [ ] No pending clearances
- [ ] No active system access
- [ ] Exit fully documented
