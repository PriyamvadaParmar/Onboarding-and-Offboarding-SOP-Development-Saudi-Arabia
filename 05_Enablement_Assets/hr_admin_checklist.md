# HR Admin Onboarding Execution Checklist

This checklist is designed for HR administrators to ensure consistent and compliant execution of the onboarding workflow inside an HRIS or manual system.

## Pre-Joining
- [ ] Offer acceptance received and recorded
- [ ] Employment contract prepared and shared
- [ ] Pre-joining communication sent to employee

## Employee Record Setup
- [ ] Employee profile created in HRIS
- [ ] Role, department, and reporting manager assigned
- [ ] Start date confirmed

## Document Management
- [ ] Mandatory documents requested
- [ ] Documents uploaded by employee
- [ ] Documents verified and approved
- [ ] Rework completed if required

## Compliance Confirmation
- [ ] Policy acknowledgements assigned
- [ ] Employee acknowledgements recorded
- [ ] Compliance requirements satisfied

## Activation
- [ ] Orientation completed
- [ ] System access initiated (if applicable)
- [ ] Employee status updated to Active

## Completion Check
- [ ] All onboarding steps completed
- [ ] No pending compliance items
- [ ] Employee ready for productive work
